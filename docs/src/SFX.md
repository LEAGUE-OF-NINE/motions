# SFX

Create a sound folder and grab the [editor script](assets/exportMotion.cs) from the button. Import the editor script into your `editor` folder, and create the `Sound` folder alongside your other folders.

Here's how to add SFX to custom motions.

{{#template templates/video.md id=assets/sfx.mp4}}

1. Select motion that you wish to add SFX to.
2. Drag and drop sound file from /sounds/ into the timeline (make sure these sounds are linked to your bundle in the bottom right after clicking the SFX.)
3. Match to the animation
4. Right click and click build asset bundle. The plugin will generate the stuff it needs so you'll see extra files in the project.
5. SFX should now play (however, it may be quiet. You may use external editing software to change the volume of the SFX.)
